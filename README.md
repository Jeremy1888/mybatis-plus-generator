**项目说明** 
- generator是项目的代码生成器，可在线生成entity、xml、dao、service，controller 减少开发任务


 **本地部署**
- 通过git下载源码
- 修改application.yml，更新MySQL账号和密码、数据库名称
- Eclipse、IDEA运行GeneratorApplication.java，则可启动项目
- 项目访问路径：http://10.39.1.47:8989
